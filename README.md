# Calender for student ( 2022-02-09 ~ 2022-02-27 )

## 팀 이름 : C버먹자!!



## 개요

대학생들의 효율적인 대학생활을 위한 과제 일정 관리 캘린더



## 주요 기능

- 과제 일정 관리 : 과제 마감 기한이 얼마나 남았는지 알 수 있음.
- 검색 기능 : 키워드를 검색하였을때, 관련 내용이 있는 해당 날짜 및 내용 출력하기



## 추가 기능 

달력출력
비밀번호 입력
해야 할 일 저장 및 검색 기능
